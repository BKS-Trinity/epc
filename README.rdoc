== README

Aby uruchomić aplikację należy wykonać kolejne kroki:
* w wierszu poleceń przechodzimy do folderu aplikacji
* tworzymy baze danych poleceniem: rake db:migrate
* uruchamiamy serwer poleceniem: rails server
* otwieramy aplikację wpisując w przeglądarce internetowej adres: http://localhost:3000/

Wymagania:
* Ruby 2.1
* Ruby On Rails 4.1.8
* SQLite3
